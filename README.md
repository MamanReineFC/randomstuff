# Wellness Deck

A gentle self-help app that draws a single wellness card — a short, guided practice — whenever you need one. Each card is a few minutes long and pulls from three traditions: **CBT**, **Somatic**, and **C-PTSD** work.

**Live app:** https://mamainreine.github.io/randomstuff/

Works in any browser, on phone or computer — no install needed.

## What it does

- **Draw a card** from the full deck or narrow to a category (Breathwork, Grounding, Self-Compassion, and more).
- **Work through it** with checkable steps, a built-in countdown timer, and a breathing pacer where relevant.
- **Track how you feel** with a before/after 0–10 scale so you can see what helped.
- **Keep a journal** of completed practices and notes.

Your journal is stored only on your own device — nothing is uploaded or shared.

## Built with

[Expo](https://expo.dev/) + React Native (react-native-web). The live site is deployed automatically to GitHub Pages on every push.

## Run it locally

```
npm install
npm run web
```

Then open http://localhost:8081.

## A note

These are gentle self-help practices — not therapy or medical advice. Go at your own pace, and stop if a practice feels overwhelming. If you're in crisis or thinking of harming yourself, please contact a crisis line or emergency services.
